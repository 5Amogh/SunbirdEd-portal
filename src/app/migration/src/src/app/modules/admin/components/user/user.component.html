Instructions :
  1. Download the csv file for reference
  2. You can add or upload details of up to 199 users at a time in one csv file
  3. The following columns are mandatory:
    1. FirstName: The user’s first name
    2. Email: The user’s verified email ID
    3. UserName: The unique name assigned to the user by the organization.Ensure that no two users of the same organization have the same username.
    4. Password: Unique or common password given by organization
  4. All other columns are optional. These columns include information about the user such as:
    1. Phone
    2. Role: User’s role on DIKSHA. Examples for this column: CONTENT_CREATOR, CONTENT_REVIEWER, FLAG_REVIEWER, COURSE_MENTOR, ORG_ADMIN, SYSTEM_ADMINISTRATION, COURSE_ADMIN, COURSE_CREATOR
    3. Grade: Classes taught by the user
    4. Location: User’s place of location of the user
    5. DOB: User’s date of birth, the value should be entered in DD-MM-YYYY format
    6. Gender: User’s gender
    7. Language: User's language preferences. Examples: English, Gujarati, Hindi, Kannada, Marathi, Punjabi, Tamil, Telugu
    8. Profile summary: User’s profile summary
    9. Subjects: Taught by the user. Examples: Assamese, Bengali, Hindi, Kannada, Malayalam, Oriya, Punjabi, Tamil, Telugu, Biology, Chemistry, Physics, Mathematics, Environmental Studies, Geography, History, Political Science, Economics, Sanskrit, Gujarati, Marathi, Nepali
    10. Columns titled emailVerified, phoneVerified and provider are conditionally mandatory. Value for emailVerified and phoneVerified should be True, if details are provided for provider
