Upload Organizations

Instructions :
  1. Download the csv file for reference
  2. You can add or upload details of up to 199 organizations at a time in one csv file
  3. The OrgName column is mandatory. Enter organization name in this column
  4. Enter each organization's name in a separate row
  5. Entering details in all other columns is optional:
    1. isRootOrg: Valid values for this column True False
    2. channel: Unique ID provided during master organization creation
    3. externalId: Unique ID associated with each organization in the administrating organization's repository
    4. provider: Channel ID of the administrator organization
    5. description: Details describing the organization
    6. homeUrl: Organization's homepage url
    7. orgCode: Organization's unique code, if any,
    8. orgType: Type of organization, such as, NGO, primary school, secondary school etc
    9. preferredLanguage: Language preferences for the organization, if any
    10. contactDetail: Organization's phone number and email ID. Details should be entered within curly brackets in single quotes. For example: ['Phonenumber': '1234567890']
    11. channel is mandatory if value for column isRootorg is True
    12. externalid and provider are mutually mandatory
