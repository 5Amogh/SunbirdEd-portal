This course uses Eclipse IDE, Maven build tool and Git for version control. You need to set these up to complete the exercises and the projects in this course.

Eclipse - Install and configure Eclipse IDE on your machine. You will need the Eclipse IDE to work with the exercises and assignments as part of the lectures.

Setting up Eclipse on Windows

Maven - Use the link below to learn how to install and configure Maven build tool. Maven is one of the most popular build tools used in the industry. It is suitable for large, complex projects and simplifies the daily development tasks such as compiling, testing and packaging your code. Ultimately, increases your productivity.

Install Maven on Windows

Git - Install and configure Git with a local repository. Git is a version control system that you can work with as a local repository without needing any server. This makes it simpler to get started with for beginners. Git is also fast becoming the most popular source control system replacing SVN.
